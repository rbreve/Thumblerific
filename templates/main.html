Make thumbnails fast!

Thumblerific is a small application built under Google App Engine that will generate thumbnails from Images hosted in other sites. Given a Image URL, Thumblerific will create a thumbnail.

Thumblerific only works with jpeg, gif and png images, it will *not* make thumbnails for web pages, for that you can use Web Snapr, Web Thumb or Thumbalizr.

Try it

Image Url: Width:
ie: http://www.site.com/image.jpg or .png or .gif

or

Upload Image: Width:

API Usage

Make instant thumbnails for your blogs, forum or any other web application just call it like this. The width parameter can be from 20px to 1200 px. In the image_url parameter include a valid Image URL.

<img src="http://thumblerific.appspot.com/t?image_url=http://www.site.com/image.jpg&width=90">
